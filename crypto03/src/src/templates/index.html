Round di permutazione

Descrizione

Prova a decifrare il messaggio. Lo abbiamo cifrato applicando le permutazioni indicate nel codice sottostante.

Questo è il codice che abbiamo usato per cifrare il messaggio: 

permutations = [
    [4, 1, 0, 3, 5, 2, 6, 7],
    [2, 6, 7, 0, 1, 3, 5, 4] ,
    [5, 2, 3, 7, 4, 0, 1, 6] ,
    [2, 6, 0, 3, 4, 7, 5, 1],
    [1, 2, 0, 4, 7, 5, 6, 3]
]


def apply_permutation(list_of_bits, permutation):
    temp=[list_of_bits[i] for i in permutation]
    return temp

def round(list_of_bits, permutation):
    temp = apply_permutation(list_of_bits, permutation)
    return temp

# value is the list of bits of the binary representation of a number
def encrypt(value, permutations):
    ROUNDS = len(permutations)
    # make the list of bits from value
    list_of_bits = [i for i in value]

    # applying permutations to list_of_bits
    for i in range(ROUNDS):
        list_of_bits = round(list_of_bits, permutations[i])
    
    # convert binary list to integer
    binary = "".join([str(i) for i in list_of_bits])
    return binary
